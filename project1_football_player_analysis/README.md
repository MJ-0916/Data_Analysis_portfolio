# Project 1 — Football Player Scores Analysis

## Overview
This project analyzes a football player scores dataset to understand
which factors are most strongly associated with player's Rating.

The analysis focuses on data quality assurance, preprocessing, and
exploratory data analysis (EDA).

## Dataset
- ~18,979 players
- 78 variables (numeric + categorical)
- Target variable: Rating (0–100)

## Structure
- `data/` — cleaned dataset
- `summary.md` — detailed analysis and insights
